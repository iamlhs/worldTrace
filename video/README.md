# 视频事件分析展示平台

这是一个纯 HTML 静态页面版本，不需要启动 Python 或 Flask。

## 使用方式

直接在浏览器打开：

```text
/data2/dyl/video_analyzer/index.html
```

也可以在文件管理器中双击 `index.html`。

## 功能

- 浏览器本地选择视频并预览
- 展示固定的视频内容摘要
- 展示固定思维导图
- 提供 3 次固定问答：每次输入一个问题，页面返回一个回答
- 播放项目内固定精彩集锦：`static/highlights/parade_highlight_30s.mp4`

## 说明

静态 HTML 页面没有后端，因此选择的视频只在浏览器本地预览，不会保存到项目目录，也不会上传到服务器。
